Search functionality with Observables

Problem Statement: If we observe while making our search, we notice each letter we type fire’s a request to the server and we don’t want it to fire a request immediately we type. We want it to delay a little before sending a request because we may be typing a long word. To handle this, we need to import debounceTime and distinctUntilChanged.

  • debounceTime : The debounceTime operator emits the latest value and helps in delaying the values transmitted by the root Observable for the specified time.
  • distinctUntilChanged : The distinctUntilChanged Returns an observable series that carries only distinguished adjacent elements according to the key selector and the compare.
  • finalize : RxJS finalize operator which will help you to stop the loader / anything that we put in finalize case in both cases, when the observable execution completes in success or when it fails.
