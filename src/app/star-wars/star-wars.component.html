NgRx Effects showcasing

Problem Statement: We are making an async calls to 3 services and showing the results on button click.

  • Effetcs: In angular ecosystem, components are reposnible for interacting with external resources through services. With NGRX Effects, it provide a way to interact with services and isolate them from components.
  • Effects are used to handle tasks such as fetching data, long-running tasks that produce multiple events and so on which components don't need to know about.
  • Effects isolate side effects from components, allowing for more pure components that select state and dispatch actions.
  • Effects are long-running services that listen to an observable of every action dispatched from the Store.
  • Effects filter those actions based on the type of action they are interested in. This is done by using an operator.
  • Effects perform tasks, which are synchronous or asynchronous and return a new action.
